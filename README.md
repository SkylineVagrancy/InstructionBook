# InstructionBook
生活中经常会遇到这种情况，有些电器或者工具，日常使用很简单，但是如果要使用一些高级功能，就必须查看说明书。但是电器或者工具一般的能使用好几年，说明书早就不知道被扔到哪里了。希望能通过这个项目，收集整理常用电器工具的使用说明，后期考虑做成一个网站和APP，免费提供给大家使用。

## 格式

* 一级目录：品牌名
* 二级目录：产品分类
* 三级目录：型号
* 四级目录：图片、MD格式的使用说明
